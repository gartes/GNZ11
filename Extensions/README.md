### Класс  выполнения заданий переданных в файле санифест
***
##### Описание тэга <PREFLIGHT />
```xml
<preflight>
<!-- Тэг для удаления файлов    -->
    <delite>
        <files>
            <!-- Имя файла от корня сайта  -->
            <filename>a-b_test_analytics.php</filename>
        </files>
    </delite>
</preflight>
```
